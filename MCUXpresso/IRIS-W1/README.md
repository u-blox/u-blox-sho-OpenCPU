# u-blox IRIS-W1 Module

This repository contains the setup information and supported files for u-blox IRIS-W1 module:

## Folder Structure

```shell
├── [EVK-IRIS-W1](EVK_IRIS-W1)
├── [extra](#extra)
├── [README.md](#readme-main)
├── [sw_config](#sw_config)
├── [tools](#tools)
└── [USB-IRIS-W1](#usb-iris-w1)
```

### EVK-IRIS-W1

Contains setup information and supported files for the EVK-IRIS-W1 module.

### extra

Contains additional tools and patches.

- IAR Segger support patch for RW610 PRC.

### sw_config

Contains software configuration files.

**flash_config files** 
- Fidelex_8MB: Fidelex 8MB flash file.
- Macronix_8MB: Macronix 8MB flash file.

### tools

**blhost**: tools for blhost operations.
**IRIS_blhost.bat**: Batch script for IRIS blhost operations.

### USB-IRIS-W1

Contains setup information and supported files for the USB-IRIS-W1 module.
